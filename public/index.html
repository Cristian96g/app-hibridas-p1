Bienvenido a la API de Lugares Turísticos

Descripción

Esta API proporciona acceso a información sobre diferentes lugares turísticos. Puedes encontrar una variedad de lugares, incluidas ciudades, sitios históricos, playas, montañas y más.

Funcionalidades

  • Obtener una lista de lugares turísticos
  • Filtrar lugares por tipo (ciudad, sitio histórico, etc.)
  • Ver las puntuaciones de los lugares
  • Ordenar los lugares por puntuación
  • Realizar operaciones CRUD en lugares y países

Endpoints Disponibles

  • GET /lugares - Obtener todos los lugares
  • GET /lugares/:id - Obtener información detallada de un lugar específico
  • GET /lugares/tipo/:tipo - Filtrar lugares por tipo
  • GET /lugares/ordenar/:orden - Ordenar lugares por puntuación
  • POST /lugares - Crear un nuevo lugar
  • PUT /lugares/:id - Actualizar un lugar existente
  • DELETE /lugares/:id - Eliminar un lugar existente
  • GET /paises - Obtener todos los países
